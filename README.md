# Eco-Share API

Backend API untuk platform penyewaan alat elektronik bekas berbasis NodeJS, ExpressJS, dan MySQL.

## Tech Stack

- NodeJS
- ExpressJS
- MySQL
- Sequelize ORM
- JWT Authentication
- bcryptjs

---

## Features

- JWT Authentication
- Role Middleware
- Item Management API
- Rental Transaction API
- Global Error Handler
- Sequelize Transaction
- Service Layer Architecture
